# 🎯 Quiz System in C++

## 📌 Description
This is a simple Quiz System project built using C++.  
The system takes user input (Name and Roll Number) and then starts a quiz based on C++ questions.

---

## 🚀 Features
- Takes user **Name** and **Roll Number**
- Asks **5 C++ questions**
- Calculates total score
- Displays **Pass/Fail result**

---

## 🧠 Passing Criteria
- Total Questions: 5  
- Each correct answer = 1 mark  

👉 **Result Rule:**
- If score **> 50%** → ✅ PASS  
- If score **< 50%** → ❌ FAIL
